# RemoveBG Free

Free online AI background remover. Deployed on Netlify.

## Deployment

Push to GitHub and connect to Netlify:

1. `git add . && git commit -m "Initial commit"`
2. Create a GitHub repo (empty, no README)
3. `git remote add origin <your-repo-url>`
4. `git push -u origin main`
5. In Netlify: "Import from Git" → select this repo → Deploy
